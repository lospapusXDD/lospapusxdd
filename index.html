🐧 Linux pa los Papus

guía tóxica de distros — pa los q quieren pasarse al lado oscuro

🟠
Ubuntu
PRINCIPIANTE

El Windows de Linux. Todo el mundo empieza acá. Si no sabés qué instalar, instalá Ubuntu y ya.

✅ Ventajas
• Facilísimo de instalar
• Enorme comunidad
• Compatible con todo
• Perfecto para empezar
❌ Desventajas
• Snapd es horrible
• Pesado en RAM
• GNOME consume mucho
• Muchos bugs en versiones nuevas
🌐 ubuntu.com
🌿
Linux Mint
PRINCIPIANTE

Ubuntu pero sin los problemas de Ubuntu. La distro más recomendada para quien viene de Windows. Simple, bonita y estable.

✅ Ventajas
• Súper fácil de usar
• Muy estable
• Sin Snapd
• Cinnamon es hermoso
❌ Desventajas
• Actualizaciones lentas
• Diseño un poco anticuado
• No tan flexible
• Basado en Ubuntu LTS
🌐 linuxmint.com
🎩
Fedora
INTERMEDIO

La distro más actualizada del mundo. Siempre tiene lo último. Es la base de Red Hat y CentOS. Seria pero moderna.

✅ Ventajas
• Software siempre actualizado
• GNOME puro y pulido
• Muy segura
• Flatpak nativo
❌ Desventajas
• Codecs no incluidos
• Actualización cada 6 meses
• dnf es lento
• Menos paquetes que Ubuntu
🌐 fedoraproject.org
🎮
Nobara Linux
⭐ RECOMENDADA GAMING

Fedora pero optimizada para gaming al máximo. La favorita actual de emilio. Hermosa, rápida y con todo incluido.

✅ Ventajas
• Gaming de primera
• KDE Plasma hermoso
• Steam y drivers incluidos
• Flatpak listo
❌ Desventajas
• Sin AUR
• Comunidad pequeña
• Basada en Fedora
• Todavía en prueba por emilio 💀
🌐 nobaraproject.org
🏛️
Arch Linux
EXPERTO 💀

La distro de los que se creen hackers. Instalación por terminal pura. Si la instalás sos automáticamente el más inteligente del grupo.

✅ Ventajas
• Control total del sistema
• AUR: instalar lo que sea
• Rapidísimo
• Rolling release
❌ Desventajas
• Instalación brutal
• Se rompe si actualizás mal
• Solo para expertos
• "Por qué sufrir" — emilio
🌐 archlinux.org
🐉
Garuda Linux
INTERMEDIO 🔥 GAMING

Arch pero bonito y con rueditas. KDE Plasma configurado para gaming. El sistema de emilio hasta que le dio problemas.

✅ Ventajas
• Diseño hermoso
• Gaming optimizado
• BTRFS + snapshots
• AUR disponible
❌ Desventajas
• Pesado en RAM
• Se rompe solo a veces
• Updates caóticos
• Demasiado bloatware
🌐 garudalinux.org
☠️
Kali Linux
HACKER 💀 SEGURIDAD

La distro que todos instalan creyendo que van a hackear al vecino. Para pentesting real. "No me gustó" — emilio 2026 💀

✅ Ventajas
• +600 herramientas de seguridad
• Perfecto para pentesting
• Documentación enorme
• Se ve cool
❌ Desventajas
• No es para uso diario
• Sin gaming decente
• Root por defecto
• "No me gustó" — emilio 💀
🌐 kali.org
🦜
Parrot OS
SEGURIDAD

Kali pero más bonito y usable en el día a día. Más ligero y con mejor diseño. Para los que quieren hacking con estilo.

✅ Ventajas
• Más ligero que Kali
• Usable en el día a día
• Mejor diseño que Kali
• Privacidad enfocada
❌ Desventajas
• Menos tools que Kali
• Comunidad pequeña
• Basado en Debian
• Menos conocido
🌐 parrotsec.org
🌀
Debian
INTERMEDIO

El abuelo confiable. Nunca se rompe. La base de Ubuntu, Kali y Parrot. Lento para actualizar pero nunca te falla.

✅ Ventajas
• Ultra estable
• La base de todo
• Liviano y eficiente
• Enorme comunidad
❌ Desventajas
• Software desactualizado
• Diseño feo por defecto
• Configuración manual
• Aburrido visualmente
🌐 debian.org
🪐
Zorin OS
PRINCIPIANTE

El Linux para los que vienen de Windows. Parece macOS, funciona fácil. Perfecto para Jero si algún día usa Linux 💀

✅ Ventajas
• Hermoso por defecto
• Facilísimo
• Estable como roca
• Soporte apps Windows
❌ Desventajas
• La versión buena es de pago
• Actualizaciones lentas
• No tan flexible
• Demasiado "seguro"
🌐 zorin.com/os
🚀
Pop!_OS
PRINCIPIANTE GAMING

Ubuntu mejorado por System76. Gaming, diseño moderno y drivers GPU incluidos. Muy popular entre gamers de Linux.

✅ Ventajas
• Drivers NVIDIA incluidos
• Gaming muy bueno
• Tiling window manager
• Diseño limpio
❌ Desventajas
• GNOME pesado
• Basado en Ubuntu
• Sin Wayland estable
• Actualizaciones lentas
🌐 pop.system76.com
🟢
Manjaro
INTERMEDIO

Arch sin el dolor. Instalador gráfico y AUR disponible. Para los que quieren Arch pero no quieren sufrir en la instalación.

✅ Ventajas
• Arch accesible
• AUR disponible
• KDE/GNOME/Xfce
• Hardware detection
❌ Desventajas
• Updates retrasados vs Arch
• Fama de romper cosas
• Certificados vencidos famoso
• No tan estable
🌐 manjaro.org
🔺
EndeavourOS
INTERMEDIO

Arch con instalador. Más limpio que Manjaro y más cercano al Arch puro. Emilio lo instaló y dijo que no le gustó el diseño 💀

✅ Ventajas
• Arch puro sin bloatware
• AUR completo
• Muy estable
• Comunidad amigable
❌ Desventajas
• Diseño básico por defecto
• Requiere configuración
• Terminal-centric
• "No me gustó" — emilio 💀
🌐 endeavouros.com
🦎
openSUSE
INTERMEDIO

La distro alemana seria. YaST es la mejor herramienta de configuración de Linux. Usada mucho en empresas y servidores.

✅ Ventajas
• YaST es increíble
• Muy estable (Leap)
• Rolling release (Tumbleweed)
• Btrfs por defecto
❌ Desventajas
• zypper es lento
• Poca popularidad
• Menos documentación
• Curva de aprendizaje
🌐 opensuse.org
🍎
elementary OS
PRINCIPIANTE

El macOS de Linux. Diseño impresionante y muy fácil de usar. Si venís de Mac este es tu Linux. Hermoso pero limitado.

✅ Ventajas
• El más bonito del mercado
• Muy fácil de usar
• Pantheon desktop único
• Buena App Center
❌ Desventajas
• Pide donación al instalar
• Muy limitado
• Pocas apps nativas
• Actualizaciones lentas
🌐 elementary.io
🪨
Rocky Linux
AVANZADO

El reemplazo de CentOS. Para servidores serios. Nadie lo usa en casa pero en empresas es el estándar. Estabilidad extrema.

✅ Ventajas
• Ultra estable
• Soporte empresarial
• Compatible con RHEL
• 10 años de soporte
❌ Desventajas
• No es para gaming
• Software muy viejo
• Solo para servidores
• Cero diversión
🌐 rockylinux.org

BlackArch
EXPERTO 💀 SEGURIDAD

Arch + 2800 herramientas de hacking. Para los que Kali les parece poco. Si llegás a instalar esto sin ayuda sos un crack.

✅ Ventajas
• +2800 herramientas
• Arch base = AUR
• Rolling release
• El más completo en seguridad
❌ Desventajas
• Instalación brutal
• Solo para expertos
• Pesa muchísimo
• No es para uso diario

Tails
PRIVACIDAD

El sistema operativo de los espías. Corre desde USB, no deja rastros y todo va por Tor. Si querés desaparecer como Samuel, usá Tails 💀

✅ Ventajas
• Anonimato total
• No deja rastros
• Tor integrado
• Arranca desde USB
❌ Desventajas
• No se instala en disco
• Sin persistencia por defecto
• Muy lento
• No sirve para gaming
🌐 tails.boum.org
🕵️
Whonix
PRIVACIDAD

Corre en máquina virtual con anonimato extremo. Tor en todo el sistema. Para los que quieren privacidad sin sacrificar el sistema principal.

✅ Ventajas
• Anonimato extremo
• Corre en VM
• Tor en todo el sistema
• No afecta al host
❌ Desventajas
• Necesita VM
• Muy lento
• Configuración compleja
• Para usuarios avanzados
🌐 whonix.org

Alpine Linux
EXPERTO

El más ligero del mundo. Pesa menos de 10MB. Usado en Docker y contenedores. No tiene systemd. Minimalismo extremo.

✅ Ventajas
• Ultra ligero (~5MB)
• Muy seguro
• Perfecto para Docker
• Arranque rapidísimo
❌ Desventajas

• Sin systemd
• musl libc incompatible
• No es para escritorio
